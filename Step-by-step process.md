To track all steps in the establishment of the pipeline

- Install VS code, Git and create repository from vs code, then publishing it as public repository on Github
- install AWS toolkit extension in VS Code
- connect to AWS (view->command palette-> AWS:Create Credentials). This installs AWS CLI.
- create access key and access key password in AWS (for root user or for IAM user)
- use it to setup connection to VSCode to AWS (create credentials, type key and password, edit credentials file, switch to pofile)
- install HashiCorp Terraform extension for syntax highlighting in VSCode
